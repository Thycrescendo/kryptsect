# KryptSect

## > A clean, beginner-friendly decentralized trading platform for GoodDollar (G$) on Celo.

![KryptSect Banner](public/images/banner.png)

### What is KryptSect?

KryptSect is a decentralized trading platform that tries to feel like a normal exchange, but runs on Web3. 

We built it because most DEXes still feel complicated for regular people. Our goal is simple: let anyone connect their wallet and start trading without needing to be a crypto expert. 

It’s built with **Next.js**, **TypeScript**, and **Tailwind CSS**, and it has native support for **GoodDollar (G$)** on the Celo network. This project is our submission for **GoodBuilders Season 4**.

---

### Why We’re Building This

A lot of decentralized exchanges are powerful but intimidating. We wanted to create something that feels familiar if you’ve ever used Binance, Coinbase, or TradingView — while staying fully decentralized.

KryptSect focuses on:

- A clean and simple interface
- Easy wallet connection (no extra steps)
- Real-time market data
- Low fees thanks to Celo
- Real utility for GoodDollar

**Our mission:** Make decentralized trading actually usable for GoodDollar users around the world.

### How KryptSect Helps GoodDollar

Instead of treating G$ as just another token you can trade, we’re making it useful across the whole platform. This should increase real usage of G$ through trading, rewards, staking, and learning.

By doing this, we hope to bring:
- More daily active wallets using G$
- Higher transaction volume
- Better onboarding for new users
- More meaningful activity in the GoodDollar ecosystem

### Main Features

**Live Markets**  
See real-time prices, trending coins, market cap, volume, and price changes. You can also create watchlists to follow assets you care about.

**Spot Trading**  
Buy and sell cryptocurrencies directly from your connected wallet. You can view market depth and place orders easily.

**Futures Trading** (coming in Phase 3)  
Trade with leverage. Choose long or short positions and manage them in one place.

**Portfolio Dashboard**  
See all your wallet balances, trading history, open positions, and overall performance in one view.

**Wallet Support**  
Works with MetaMask, WalletConnect, Coinbase Wallet, and other RainbowKit-compatible wallets.

**Mobile Friendly**  
The interface works well on desktop, tablet, and mobile.

### GoodDollar Integration

This is where KryptSect gets interesting for the GoodDollar community:

- **Trade G$ pairs** — G$/USDT, G$/CELO, G$/ETH, G$/BTC and more
- **Pay fees in G$** — Get discounted trading fees when you pay with GoodDollar
- **Earn G$ rewards** — For trading, providing liquidity, and participating in the community
- **Liquidity Mining** — Supply liquidity to pools and earn G$ rewards
- **Stake G$** — Stake to get lower fees, better analytics, and other perks
- **Referral Program** — Invite friends and earn G$ when they trade
- **Learn-to-Earn** — Complete short lessons and quizzes about trading and DeFi, and earn G$ for finishing them

### What We Used to Build It

**Frontend**
- Next.js 15
- React + TypeScript
- Tailwind CSS

**Wallet & Blockchain**
- RainbowKit + Wagmi + Viem
- Celo network
- GoodDollar contracts

**Charts & Data**
- Lightweight Charts
- CoinGecko API

We’re also planning to add a backend (Node.js + PostgreSQL + Redis) and smart contract features later.
